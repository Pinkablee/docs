p-blocks is a FiveM script that allows you to create customizable gang locations and blocks in your server.

**Features:**
- Customizable gang blocks with labels, colors, and sizes.
- Unlimited gang locations for complete customization.

**Dependencies:**
- This script requires the **[ox_lib](https://github.com/overextended/ox_lib)** library.

**Price:** Free
