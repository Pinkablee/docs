title: 'Configuration Guide'
---

### Screenshot Delay
- Adjust the frequency of admin screenshots in seconds. After taking a screenshot, a cooldown period will precede the next one. Recommended range: 5-10 seconds.

### Report Delay
- Set the cooldown duration for player reports in seconds. After submitting a report, there will be a cooldown before making another. Recommended range: 5-10 seconds.

### Console Info (Print to Server Console)
- Opt to display admin menu information in your server console. This includes player actions such as kicks, bans, unbans, and various other details when banned players attempt to join. Enable (true) or disable (false) this option.

### Discord Bot Token
- Enter the bot token of a Discord bot within your server. The bot must have permission to view members and roles. This field is mandatory.

### Discord Guild ID
- Provide the Guild ID of your Discord server where the bot is active. Ensure the Guild ID matches the server the bot is a member of. This field is mandatory.

### Discord Roles
- Define player permissions based on Discord roles. Create a table with each role as a separate entry. Players possessing specific roles in your Discord server will be granted corresponding permissions.
