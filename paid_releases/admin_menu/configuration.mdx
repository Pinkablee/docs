---
title: 'Configuration Guide'
---

Adjust the frequency of admin screenshots in seconds. Cooldown: 5-10 seconds.
```lua cfg.lua
cfg.screenshotDelay = 5 -- (type: number)
```

Set the cooldown for player reports in seconds. Cooldown: 5-10 seconds.
```lua cfg.lua
cfg.reportDelay = 5 -- (type: number)
```

Display admin menu info in the server console. Enable (true) or disable (false).
```lua cfg.lua
cfg.consoleInfo = true -- (type: boolean)
```

Activate debug mode.
```lua cfg.lua
cfg.debug = true -- (type: boolean)
```

Discord webhooks.
```lua cfg.lua
cfg.logs = {
    kicks = {enabled = true, webhook = 'YOUR_WEBHOOK_HERE'},
    bans = {enabled = true, webhook = 'YOUR_WEBHOOK_HERE'},
    offlineBans = {enabled = true, webhook = 'YOUR_WEBHOOK_HERE'},
    unbans = {enabled = true, webhook = 'YOUR_WEBHOOK_HERE'},
    banLengthEdited = {enabled = true, webhook = 'YOUR_WEBHOOK_HERE'},
    banReasonEdited = {enabled = true, webhook = 'YOUR_WEBHOOK_HERE'},
    reports = {enabled = true, webhook = 'YOUR_WEBHOOK_HERE'},
    claimedReports = {enabled = true, webhook = 'YOUR_WEBHOOK_HERE'},
    deletedReports = {enabled = true, webhook = 'YOUR_WEBHOOK_HERE'},
    freezed = {enabled = true, webhook = 'YOUR_WEBHOOK_HERE'},
    staffChat = {enabled = true, webhook = 'YOUR_WEBHOOK_HERE'},
    screenshots = {enabled = true, webhook = 'YOUR_WEBHOOK_HERE'},
    identifiers = {
        steam = true,
        license = true,
        license2 = true,
        xbox = true,
        live = true,
        fivem = true,
        discord = true
    }
} -- (type: table)
```

Enter the Discord bot's token. Mandatory field.
```lua cfg.lua
cfg.discord.botToken = "YOUR_BOT_TOKEN_HERE" -- (type: string)
```

Provide the Guild ID of your active Discord server. Mandatory field.
```lua cfg.lua
cfg.discord.guildId = "YOUR_GUILD_ID_HERE" -- (type: string)
```

Define player permissions based on Discord roles.
```lua cfg.lua
cfg.discord.roles = {
        {
            roleId = YOUR_ROLEID_HERE, 
            perms = {
                immune = true, 
                open = true, 
                kick = true, 
                ban = true, 
                offlineban = true, 
                unban = true, 
                editban = true, 
                teleport = true, 
                spectate = true, 
                freeze = true, 
                screenshot = true, 
                invisibility = true, 
                godmode = true, 
                noclip = true, 
                nametags = true, 
                wipe = true, 
                deletereport = true, 
                claimreport = true, 
                staffchat = true
            }
        },
    }
} -- (type: table)
```
