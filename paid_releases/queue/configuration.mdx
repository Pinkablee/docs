---
title: 'Configuration Guide'
---

The time, in seconds, it takes for the queue to load when first initiated.
```lua cfg.lua
cfg.joinDelay = 60 -- (type: number)
```

Set to true to display the queue status in the server's hostname.
```lua cfg.lua
cfg.displayQueueInHostName = true -- (type: boolean)
```

Display queue information on the server console? (enabled by default)
```lua cfg.lua
cfg.consoleInfo = true -- (type: boolean)
```

Enable Discord authentication on the server console to ensure all data is being retrieved. (enabled by default)
```lua cfg.lua
cfg.debug = true -- (type: boolean)
```

Full points from all roles if true, only highest role's points if false.
```lua cfg.lua
cfg.stackPrio = true -- (type: boolean)
```

Configuration for Discord integration.
```lua cfg.lua
cfg.discord = {
    requireIdentifier = true, -- (type: boolean) Require Discord linking?
    whitelist = false, -- (type: boolean) Enable server whitelist?
    whitelistRoleId = YOUR_WHITELISTED_ROLE_ID_HERE, -- (type: number) Role for whitelist access.
    botToken = 'YOUR_BOT_TOKEN_HERE', -- (type: string) Bot token.
    guildId = 'YOUR_GUILD_ID_HERE', -- (type: string) Discord server ID.
} -- (type: table)
```

Configuration for priority roles.
```lua cfg.lua
cfg.prioRoles = { -- (type: table) prioRoles: Configuration for priority roles.
    {label = 'Founder', roleId = ROLE_ID_HERE, points = 10000}, -- (type: table) Founder role configuration.
    {label = 'Head Developer', roleId = ROLE_ID_HERE, points = 5000}, -- (type: table) Head Developer role configuration.
} -- (type: table)
```
