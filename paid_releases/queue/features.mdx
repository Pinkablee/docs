### Features
- Manage player priority through discord roles.
- When using discord roles, ensure you have a Discord server and a bot with necessary permissions to view members and roles.
- Access an API for querying queue information and adding/removing priority.
- Priority points stack for players, allowing multiple points from different categories to be active concurrently.
- Configure a join delay to hold players in the queue for a set duration during server restarts, ensuring scripts finish initializing.
- Assign specific priority points for each discord role using their role IDs.
- Customize queue language and displayed messages.
- Optionally deny connecting players not in your discord server.
- Utilizes adaptive cards for displaying information in a clean and modern format.
- Customize adaptive cards, including banner images, button labels, URLs, icon images, and colors.
