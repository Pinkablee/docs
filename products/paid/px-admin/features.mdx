px_admin is built as a full-featured administrative toolkit designed to give staff reliable moderation tools while maintaining security and performance.

## Core Moderation

px_admin includes powerful moderation tools accessible through the admin interface.

- Kick players with custom reasons  
- Ban players (temporary or permanent)  
- Offline ban support  
- Unban players  
- Edit existing bans  
- Freeze players  
- Kill players  
- Revive players  
- Spectate players  
- Teleport to players  
- Bring players to your location  

All moderation actions are permission-protected and logged when webhooks are configured.

---

## Reports System

The built-in report system allows players to request staff assistance.

- Players can create reports  
- Staff can assist reports  
- Reports can be closed when resolved  
- Admin session teleport support  
- Optional logging for all report activity  

This allows staff to respond quickly while keeping moderation organized.

---

## Ban Management

px_admin includes a complete ban management system.

- Temporary bans  
- Permanent bans  
- Offline bans  
- Ban editing  
- Ban expiration tracking  
- Automatic rejection on connection  

All bans include metadata such as:

- Admin name  
- Ban reason  
- Expiration time  
- Discord identifier  

---

## Admin Utilities

Staff utilities improve workflow and control.

- Toggle godmode  
- Toggle invisibility  
- Toggle nametags  
- Screenshot capture  
- Entity wipe tools  
- Cooldown protection  

These tools help maintain control without risking abuse.

---

## Permission System

px_admin supports flexible permission management.

- Discord role permissions  
- ACE permission fallback  
- Permission immunity levels  
- Fine-grained action permissions  

This allows full customization of staff roles.

---

## Logging System

px_admin supports detailed Discord webhook logging.

- Kick logs  
- Ban logs  
- Report logs  
- Screenshot logs  
- Action logs  

Logs provide accountability and staff transparency.

---

## Performance Design

px_admin is optimized for performance.

- Server-side validation  
- Lightweight permission checks  
- Cooldown protection  
- Minimal background loops  

The system remains stable even on large servers.
